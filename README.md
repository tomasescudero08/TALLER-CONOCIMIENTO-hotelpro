- HOTELPRO -

- Descripción del proyecto -

Este proyecto es una base de datos para gestionar un hotel.
Permite manejar información de habitaciones, clientes, reservas, empleados, servicios y eventos.

También incluye consultas, vistas y procedimientos almacenados para automatizar procesos como reservas, check-in, check-out y registro de consumos.

- Instrucciones de uso -

1. Crear y usar la base de datos:
2. Ejecutar las tablas e insertar datos.
3. Crear las vistas y procedimientos almacenados.
4. Probar los procedimientos con CALL.

- Estructura del sistema -

*Hoteles: información general del hotel
* Habitaciones: tipos y estado
* Clientes: datos de los huéspedes
* Reservas: registro de estancias
* Check-in / Check-out: entradas y salidas
* Servicios: consumos adicionales
* Eventos: organización de eventos y salones
* Vistas: consultas resumidas
* Procedimientos: automatizan procesos importantes
